# Living Faith Church Website

A modern, responsive website for Living Faith Church built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui components.

## Features

- Modern responsive design with Tailwind CSS
- Interactive components using shadcn/ui
- Real-time live stream integration
- Event and sermon management
- Prayer request submission
- Testimonies and gallery sections
- WhatsApp integration for FAQs
- Smooth animations with Framer Motion
- 3D visuals with Three.js

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
